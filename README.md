# GMATube — Seeder Daemon

Daemon desktop ligero para usuarios que quieren contribuir a la red GMATube de forma continua. Seedea torrents en background respetando límites de disco y ancho de banda configurados por el usuario.

**Stack:** Go · anacrolix/torrent · systray
